About SemanticPath

SemanticPath is a Semantic Web-powered Career Path & Skill Gap Analyzer built for the TSW6223 course. It uses structured XML data validated by XSD, converted into RDF/RDFS triples, and queried using SPARQL to provide intelligent career recommendations.

XML XSD RDF/RDFS SPARQL Python Flask

Career data is sourced from O*NET OnLine occupation profiles. The system maps informal career names to official O*NET occupation codes.
